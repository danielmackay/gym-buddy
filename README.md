## Quick Start

Run the entire stack (backend + database + frontend) from the repository root:

```bash
aspire run
```

This will:
- Start the backend API with Aspire dashboard
- Provision and run SQL Server database
- Run database migrations and seeding
- Start the Next.js frontend dev server
- Open Aspire Dashboard for observability

### Access Points
- **Frontend**: http://localhost:3000
- **Backend API**: https://localhost:7255
- **API Documentation**: https://localhost:7255/scalar/v1
- **Aspire Dashboard**: Opens automatically
